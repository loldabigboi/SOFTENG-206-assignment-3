# SOFTENG-206-assignment-3
Assignment 3 for SOFTENG 206

TARGET AUDIENCE
------------
18-25 year olds with English second language

PROJECT SET UP
------------
This program is intended to execute using command line in the Ubuntu VirtualBox image. Firstly change the directory to the location of the VARpedia.sh file. From command line execute "bash VARpedia.sh" to run this program.
